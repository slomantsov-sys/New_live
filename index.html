Для синхронизации между устройствами нужен бесплатный Firebase проект (Google)
Как получить конфиг (5 мин):
1. Открой console.firebase.google.com
2. Создай проект → Add app → Web (</>)
3. Скопируй apiKey, projectId, appId
4. Authentication → Sign-in → Anonymous → Enable
5. Firestore → Create database → Start in test mode
